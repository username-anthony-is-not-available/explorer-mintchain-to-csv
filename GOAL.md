# Vision: explorer-mintchain-to-csv

> One-click export of MintChain transactions to Koinly-compatible CSV for tax reporting.

## Target Users

- **Primary**: MintChain users needing tax-ready transaction exports
- **Secondary**: Crypto accountants working with MintChain portfolios

## Strategic Priorities (2025)

1. Maintain compatibility with MintChain Explorer API changes
2. Add token transfer categorization (swaps, transfers, bridges)
3. Support automatic Koinly import labels
4. Add batch processing for multiple wallets

## In Scope

- MintChain Explorer API integration
- Transaction, token transfer, and internal transaction fetching
- Koinly Universal CSV format output
- Date range filtering
- JSON export option

## Out of Scope

- Other blockchain networks (MintChain focus)
- Direct Koinly upload integration
- Real-time transaction monitoring

## Success Metrics

| Metric                        | Current | Target               |
| ----------------------------- | ------- | -------------------- |
| Transaction types supported   | 3       | 5+ (include bridges) |
| Koinly compatibility          | Basic   | Full label support   |
| Multi-wallet batch processing | No      | Yes                  |

## Competitive Positioning

- **Primary competitors**: Generic blockchain CSV exporters
- **Differentiation**: MintChain-specific, Koinly-optimized output, simple CLI
